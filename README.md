# Axom Cars Landing Page

A premium, interactive landing page for Axom Cars, featuring a modern design, dynamic vehicle showcase, and a robust lead management system.

## 🚀 Features

- **Interactive Vehicle Range**: Browse through various car models (Electric, SUV, Hatchback, Sedan) with dynamic color switching.
- **Lead Management System**: Integrated inquiry form that captures lead data in `localStorage` with a dedicated Admin Dashboard.
- **Responsive Design**: Fully optimized for mobile and desktop viewing.
- **Admin Dashboard**: View, manage, and export lead data via CSV.
- **Dynamic Content**: Sections for Showrooms, Workshops, Finance, and Careers.

## 🛠️ Technologies Used

- **HTML5/CSS3**: Custom vanilla CSS for a premium automotive aesthetic.
- **JavaScript**: Interactive elements, tab switching, and data handling.
- **Local Storage**: For client-side lead data persistence.

## 📁 Project Structure

- `index.html`: Main landing page.
- `cars.html`: Detailed vehicle range.
- `admin.html`: Admin dashboard for lead management.
- `showrooms.html` / `workshops.html`: Location-based information.
- `main.js`: Core logic for interactions and data handling.
- `style.css`: Modern, high-contrast design system.

## 📝 Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/estheticengine/axomcars.git
   ```
2. Open `index.html` in your browser.
